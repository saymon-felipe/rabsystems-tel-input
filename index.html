Rabsystems tel input

Input de telefone personalizado da empresa Rabsystems

Instalação

Primeiro baixe e inclua no seu projeto o JQUERY.

Em seguida baixe os arquivos CSS e JS abaixo e inclua na sua página HTML.

  • CSS rabsystemsTelInput.css v1.0
  • JS rabsystemsTelInput.js v1.0

O arquivo CSS deve ser declarado no HEAD do HTML, enquanto o arquivo JS deve ser declarado no BODY após o JQUERY.

Depois o seguinte código HTML deverá ser escrito para que o input funcione:

<div class="rabsystems-input">
     <div class="flag-input">
        <div class="current-flag-container"></div>
        <div class="flag-list"></div>
    </div>
    <input type="tel" name="tel" id="tel-input">
</div>
